Создать проект.
Скопирововать файлы из репозитория в проект.

Установить зависисимости из requirements.txt
pip install -r requirements.txt

В файле settings.py есть возможность выбрать браузер запуска и режим работы. По умолчанию 'chrome' и 'headless'.

Запустить тестирование.
